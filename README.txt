Ray Tracer
Bridget Winn

To run:
   create directory for .pov files - "pov"
   create directory for .tga files - "tga"
   run make
   ./raytrace <width> <height> ___.pov <destination directory>

The program will name the .tga file after the incoming .pov file
and place it in the tga directory.

Part 1:
- ray/sphere intersections
- ray/plane intersections
- basic colors

Part 2:
- Blinn Phong shading
- shadows (upcoming)
- moving camera (upcoming)
